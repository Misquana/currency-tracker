# Currency Tracker

Трекер курсов валют ЦБ РФ. CLI + REST API + веб-интерфейс.

## Требования

- Node.js 24.15.0 (используй `nvm use`)

## Установка

```bash
npm install
```

## Скрипты

| Скрипт | Что делает |
|--------|------------|
| `npm run lint` | Проверка кода линтером |
| `npm run lint:fix` | Автоматическое исправление |

## Структура

- `src/` — исходный код приложения
- `data/` — база данных SQLite (создаётся автоматически)
- `playground/` — учебные упражнения по JS

## База данных

SQLite-файл создаётся автоматически в `data/rates.db` при первом запуске.

Схема: см. `src/schema.sql`.

Чтобы посмотреть содержимое — открой файл в VS Code (расширение SQLite Viewer).
